# daimon-news

NOTE: The documentation is in progress.

## Requirements

- Ruby
- PostgreSQL
- MeCab

## [WIP] Getting Started (with docker)

1. Get code

       git clone https://github.com/daimon-news
       cd daimon-news

2. Boot server

       docker-compose up

3. Run setup script

       docker-compose exec daimon-news-web /exec bin/setup

4. Access localhost:3000
